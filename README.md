LEAD CAPTURE CHATBOT

A simple chatbot that collects user details and sends them to Google Sheets.

-----------------------
SETUP INSTRUCTIONS
-----------------------

1. Open index.html
2. Find this line:
   webhookURL: "PASTE_YOUR_WEBHOOK_URL_HERE"

3. Replace it with your Google Apps Script Web App URL

4. Save the file

5. Open index.html in your browser

-----------------------
WHAT IT DOES
-----------------------

- Collects name, phone, and requirements
- Sends data to your Google Sheet
- Works without any backend

-----------------------
WHO CAN USE THIS
-----------------------

- Real estate agents
- Freelancers
- Small businesses
- Students

-----------------------
NOTES
-----------------------

- You can edit questions directly in the code
- No installation required
